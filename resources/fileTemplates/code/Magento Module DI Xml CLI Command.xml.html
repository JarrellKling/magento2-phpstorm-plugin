Each CLI Command must be register in the related module di.xml. Declare your Command class in Magento\Framework\Console\CommandListInterface and configure the command name using dependency injection
Create a new CLI Command in Magento.
Predefined variables explanation:
${CLI_COMMAND_DI_NAME}   A CLI Command name used for DI to the CommandListInterface.
${CLI_COMMAND_CLASS}   A class of the CLI Command.
