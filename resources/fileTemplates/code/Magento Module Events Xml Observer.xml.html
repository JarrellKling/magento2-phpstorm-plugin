Observers are a certain type of Magento class that can influence general behavior, performance, or change business logic. Observers are executed whenever the event they are configured to watch is dispatched by the eobservent manager.
A observer is declared (registered) for a class in the di.xml file.
Read more about Observer, including when those can be used.
Predefined variables will take the following values:
${EVENT_NAME}   The name of the observer for the event definition.
${OBSERVER_NAME}   Observer name.
${OBSERVER_CLASS}   The fully qualified class name of the observer.
